Hướng dẫn nạp firmware

  1. 1 Cắm ESP32-S3 vào máy tính qua cáp USB
  2. 2 Nhấn nút "Kết nối và nạp firmware" bên trên
  3. 3 Trình duyệt sẽ yêu cầu chọn cổng Serial - chọn cổng có tên "USB JTAG/serial debug unit" hoặc tương tự
  4. 4 Chọn file firmware (định dạng .bin) từ máy tính của bạn
  5. 5 Quá trình nạp sẽ tự động bắt đầu và hiển thị tiến độ

Lưu ý quan trọng

• Đảm bảo ESP32-S3 ở chế độ nạp (nhấn nút BOOT khi kết nối hoặc nối chân GPIO0 xuống GND)

• Chỉ hoạt động trên trình duyệt hỗ trợ Web Serial API: Chrome 89+, Edge 89+, Opera 76+

• Cần kết nối HTTPS để hoạt động (GitHub Pages cung cấp HTTPS)
